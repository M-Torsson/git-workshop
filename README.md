\# Git Workshop



Practicing Git \& GitHub basics.



\## Commands I used



\- git init : Initialize a new local Git repository.

\- git add . : Stage all files.

\- git commit -m "message" : Save changes with a message.

\- git remote add origin URL : Connect local repository to GitHub.

\- git branch -M main : Rename branch to main.

\- git push -u origin main : Push project to GitHub.

\- git push : Push new commits.

\- git status : Show repository status.

\- git clone URL : Clone a repository.

\- git checkout -b branch-name : Create new branch.

\- git merge branch-name : Merge branch into main.

\- git revert HEAD : Revert last commit.

\- git reset --hard HEAD~1 : Remove last commit.

\- git tag v1.0 : Create version tag.
